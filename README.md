# 📚 Smart Study Planner

A smart study planner designed for Class 11–12 students to help manage subjects, stay consistent, and improve study efficiency.

---

## 🚀 Features

- 📅 Automatic study plan generation
- 🧠 Mood-based planning (Tired / Normal / Motivated)
- 🔥 Streak tracking system
- 📊 Progress visualization (graph)
- 🎯 Weak subject detection
- 📌 Today’s Focus system

---

## 💡 Why I Built This

Students often struggle with consistency and planning.  
This project helps break down study goals into manageable daily tasks while adapting to the user's energy and performance.

- Chart.js
